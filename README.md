🌟 Overview

HerRise Network is a responsive landing page built to showcase the mission, vision, values, team, and contact options of the HerRise initiative. It empowers young women by providing visibility into education, mentorship, community support, and mental health resources.

The landing page includes:

Animated hero slideshow

Mission, Vision & Values cards with hover background images

Interactive timeline

Team section with member profiles

Contact form integrated with Formspree

Smooth scrolling and modern animations

Built with HTML, CSS, and JavaScript, ready for deployment as a static site.

⚙️ Installation

Clone the repo

git clone https://github.com/<your-username>/HerRiseNetwork.git
cd HerRiseNetwork

Open locally
Open index.html in your browser.

No extra dependencies required.

💡 Usage

Hover over Mission, Vision & Values cards to reveal background images.

Scroll to see timeline animations.

Submit messages via the Contact form using Formspree.

Works on desktop, tablet, and mobile devices.

Formspree setup:

<form id="contact-form" action="https://formspree.io/f/xyknjgzj" method="POST">

Change the Formspree URL to your own if needed.

🌐 Deployment
Hosting on Vercel

https://herrisenetwork.vercel.app/


Push your repo to GitHub:

git add .
git commit -m "Initial commit"
git push origin main

On Vercel
:

Import project from GitHub

Set Framework Preset to Other

Leave Build Command & Output Directory empty

Click Deploy

Updates pushed to GitHub will redeploy automatically.
